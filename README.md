# 큠린더 (Kiwoom Calendar)

(링크: https://heroesdiary.onrender.com/)

## 개발 동기
**히어로즈 공식어플 서비스가 종료된 뒤 익명으로 직관 기록 일기장이 생기면 어떨 것 같냐는 질문을 보고 만들게 됨.**

*(현재 히어로즈 기록만 불러오지만 추후 전구단으로 확장할 의향이 있음)*
